# Project Instructions — Block Rush

## Goal
Build a premium mobile-first 8x8 block puzzle game inspired by classic block placement puzzle games, but with original branding, original visuals, original UI, and no copied assets.

## Core Game Rules
- 8x8 board.
- Player receives 3 block pieces at a time.
- Drag pieces onto the board.
- Clear full rows and columns.
- Generate 3 new pieces only after all 3 current pieces are placed.
- Game ends only when none of the current pieces can fit.
- No rotation in version 1.
- Save progress with localStorage.

## Must Feel
- Very smooth on mobile.
- Addictive, fast, colorful, satisfying.
- Premium casual puzzle game feel.
- Simple enough to understand in 5 seconds.
- Reward the player every few seconds through score, combos, coins, XP, chests, effects, or missions.

## Addictive Systems
Include:
- Combo multiplier.
- Coins.
- XP and level system.
- Chest progress and reward modal.
- Daily missions.
- Unlockable skins/themes.
- Power-ups: Hammer, Shuffle, Bomb.
- Local high score.
- Game over screen with restart and reward summary.

## UI Style
- Mobile 9:16 layout.
- Rounded cards.
- Gradient background.
- Clean spacing.
- Big readable text.
- Smooth animations.
- Satisfying particles and clear effects.
- Do not make the UI crowded.

## Technical Stack
- React + Vite.
- Clean component structure.
- localStorage for save data.
- No paid assets.
- No copyrighted assets.
- No broken buttons.
- No console errors.
- Must run with:
  - npm install
  - npm run dev
  - npm run build

## Quality Rules
Before finishing, test:
- Placing pieces.
- Invalid placement.
- Row clear.
- Column clear.
- Multiple line clear.
- Game over detection.
- Score.
- Coins.
- XP and level up.
- Chest opening.
- Missions.
- Skins.
- Power-ups.
- Restart.
- Mobile layout.
