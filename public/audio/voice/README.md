# Block Rush voice clips

Place optional recorded reward voice files in this folder and list their filenames in `voice-manifest.json`.

Recommended examples:

- `good_01.mp3`
- `great_01.mp3`
- `awesome_01.mp3`
- `amazing_01.mp3`
- `excellent_01.mp3`
- `unbelievable_01.mp3`
- `board_clear_01.mp3`
- `board_clear_02.mp3`
- `fever_01.mp3`

Multiple filenames per manifest key are randomized. Empty entries safely use the browser voice fallback.
